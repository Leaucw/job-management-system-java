# Job Management System (Java)

## Overview
This is a console-based Job Management System developed using Java.
The system allows employers to create, update, remove, and filter job postings.

## Features
- Create job postings
- Update existing jobs
- Remove jobs
- Filter jobs by:
  - Salary range
  - Location
  - Company
  - Job type
- Uses Object-Oriented Programming (OOP) principles

## Technologies
- Java
- OOP (Encapsulation, Abstraction)
- Data Structures (ArrayList / LinkedList)
